# YellowPages website

Frontend for YellowPages: Vite + React, single index route, base layout.

## Run locally

From the `website/` directory:

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173). The dev server hot-reloads on file changes.

## Build

From the `website/` directory:

```bash
npm install
npm run build
```

Output is written to `website/dist/`. To preview the production build locally:

```bash
npm run preview
```

This serves the built assets (default [http://localhost:4173](http://localhost:4173)).
